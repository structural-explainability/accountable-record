# Governance

Governance source data defines AR's scope, boundaries, and non-goals.

Authoritative source data lives in:

```text
data/governance/
```

## Purpose

Governance data explains what AR is responsible for and what AR intentionally
does not decide.

It keeps AR focused on accountable record structure rather than domain
adjudication or operational deployment.

## Governance Source Files

Governance includes:

- scope;
- boundaries;
- non-goals.

## Boundary

Governance data does not replace conformance checks.

It defines the contract boundary within which conformance, verification,
exports, packages, and profiles operate.
